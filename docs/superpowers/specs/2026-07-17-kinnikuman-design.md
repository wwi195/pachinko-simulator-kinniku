### 通常時の当たり演出フロー

```
先バレ発生
  ↓
レバブル演出（発生/非発生の2択、仮確率30%で発生）※発生しなくても当たることがある
  ↓
Vフラ演出（発生/スキップの2択、仮確率15%で発生）
  ↓
大当たりジャッジ画面
  ↓ 95%で当たり確定（レバブル・Vフラの発生有無に関わらず抽選対象）

  ↓
「RUSHチャレンジ」ボタンをユーザーが押下
  ↓ 50%で成功判定
  ├ 成功 → RUSH突入（ST145回転からスタート）
  └ 失敗 → 通常時へ戻る
```

## RUSH中の仕様（電サポ／ST型）

- ST（電サポ）: 145回転
- 1回転ごとに **1/92.3** で「発展」判定
  - ST残り回転数以内に発展しなければ RUSH終了 → 通常時へ戻る
  - 発展した場合 → 専用モードへ突入

### 専用モード（発展後）

```
専用モード突入演出
  ↓
保留を5個貯める演出
  ↓
「ジャッジ」ボタンをユーザーが押下
  ↓
5個それぞれを個別に1/2.8で当落判定（0〜5個が当たり、独立試行）
  ↓
当たり数に応じて結果表示・出玉付与:
  0個 → 「はずれ」
        → 50%でSTが145回転にリセット
        → 50%で発展前の残りST回転数から再開（進行度を維持）
  1個 → 「1500ボーナス！」（実質+1400発）
  2個 → 「3000ボーナス！」（実質+2800発）
  3個 → 「4500ボーナス！」（実質+4200発）
  4個 → 「6000ボーナス！」（実質+5600発）
  5個 → 「7500ボーナス！」（実質+7000発）
  ↓（1〜5個当たりの場合は共通）
STは145回転にリセットされRUSH継続
```

### RUSH中の回転ボタン

- 1回転
- 10回転
- 次発展まで一括スキップ（発展 or ST消化のいずれかで自動停止）
